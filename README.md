##  Projet Principal : Football-dashboard (Python) 
Ce projet est un tableau de bord complet dédié aux 5 grands championnats européens (Ligue 1, Premier League, Liga, Serie A, Bundesliga).

##  Fonctionnalités
- **Stats en temps réel** : Analyse des performances par équipe et par joueur.
- **Classement Dynamique** : Visualisation en direct de l'état des championnats.
- **Moteur de Simulation** : Algorithme de calcul pour simuler les résultats des matchs restants basé sur les statistiques de la saison.
